Get the Jolts on Your Vim

Vim is an advanced text editor that seeks to provide the power of the de-facto Unix editor 'Vi', with a more complete feature set. It's useful whether you're already using vi or using a different editor. And Vim is possible to extend the functional of editing with plugin called 'vimscript'.

However, it is not provided to unify/concise way to install thems.

But that become Legend...

VimJols provide the easy way to install 'vimscript'.

search:
# jolt search html
install:
# jolt install zencoding.vim
